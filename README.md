# FlappyBird 项目说明

## 项目简介
一个基于 C 语言 + EasyX 图形库开发的经典 FlappyBird 小游戏，还原小鸟跳跃、管道障碍、得分统计等核心玩法，适合编程初学者学习图形界面开发与游戏逻辑实现。

## 开发环境
- 编程语言：C 语言
- 开发工具：Visual Studio（推荐 2019/2022 版本）
- 依赖库：EasyX 图形库（用于绘制游戏窗口、角色与动画）
- 运行系统：Windows 10/11（EasyX 库暂不支持 macOS/Linux）

## 安装与运行
1. 安装 Visual Studio 并配置 C 语言编译环境；
2. 下载 EasyX 图形库（官网：https://easyx.cn/），按教程完成安装与环境配置；
3. 克隆本仓库到本地：`git clone https://github.com/hades88664/FlappyBird.git`（或直接下载仓库压缩包解压）；
4. 用 Visual Studio 打开项目文件夹中的工程文件（`.sln` 后缀）；
5. 点击「运行」按钮（或按 `F5` 键），即可启动游戏。

## 游戏玩法
- 按空格键控制小鸟向上跳跃，松开则自由下落；
- 躲避随机生成的上下管道障碍，不可触碰管道或窗口边界；
- 每成功穿过一组管道，自动累计 1 分；
- 触碰障碍后游戏结束，显示最终得分，按空格键重新开始。

## 项目结构
```
FlappyBird/
├── Project17/          // 项目工程文件夹
│   ├── Game.cpp        // 游戏核心逻辑（主函数、绘图、碰撞检测）
│   ├── Bird.h          // 小鸟角色相关定义（位置、跳跃逻辑）
│   ├── Pipe.h          // 管道障碍相关定义（生成、移动）
│   └── Resource/       // 游戏资源（图片、音效，可选）
└── .gitignore          // Git 忽略文件配置（过滤编译产物、临时文件）
```

## 核心功能实现
1. 图形界面初始化：通过 EasyX 库创建游戏窗口，设置背景色；
2. 小鸟动画：循环绘制小鸟图片，实现跳跃与下落的视觉效果；
3. 管道生成：随机生成管道高度与间距，定时向左移动模拟场景滚动；
4. 碰撞检测：判断小鸟与管道、窗口边界的距离，触发游戏结束逻辑；
5. 得分系统：记录穿过管道的次数，实时在窗口显示得分。

## 注意事项
- 若运行时提示「找不到 EasyX 相关头文件」，请检查 EasyX 库是否正确安装，或手动在项目中配置库路径；
- 游戏资源（图片、音效）需自行放置到 `Resource` 文件夹，并在代码中修改资源路径；
- 若需要移植到其他系统，需替换 EasyX 库（如使用 SDL 库替代）。
